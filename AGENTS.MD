Making a dekstop APP, of an AI name SOLICE. with personality that talks to user about anything.

SOLICE is designed as a desktop assistant that can be called upon by using a keyboard shortcut, 
and it will show up.SOLICE is a 3D model of a donut shape, that always facing the user's mouse cursor. with a soft blue backlit around the edge of the disk. 

SOLICE is created to be helpful for the User. When the user is typing a message, SOLICE will show up in the chat section, and it will respond to the user's message.

SOLICE can do a lot of things, such as:
- Responding to user's message
- Showing a 3D model of a donut shape
- Blinking or pulsing colors around the edge of the disk
- Talking to the user
- Answering user's questions
- Helping user with tasks
- Taking note
- Setting timers/reminders
- Describing an image.

Features :
- Text chat
- Voice chat
- Image chat
- Video chat
- Chat creation and deletion
- Chat history
- Hovering Screen share analysis : SOLICE will hover over the user's screen and can see the user screen to analyze anything on screen that they want to analyze. (NOT PRIORITY TO BE DEVELOPED AT FIRST).
